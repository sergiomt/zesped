¿Cómo funciona el soporte multiusuario y multiempresa?

Para poder empezar a utilizar Zesped se requiere crear en primer lugar una Cuenta de Cliente. Esta cuenta de cliente debe contener, al menos, un usuario propietario, y al menos un obligado fiscal. En cada cuenta de cliente puede haber más usuarios y más obligados fiscales. Las facturas y las notas de almacenan asociadas a un obligado fiscal (el receptor de la factura).

Proveedores

Cada cuenta de cliente tiene su propio listado de proveedores. Los proveedores de una cuenta cliente son compartidos por todos sus obligados fiscales.

Empleados

Cada obligado fiscal puede tener una lista de empleados. Los empleados son requeridos como cobradores de las notas de gasto liquidadas. En el módulo de digitalización de facturas no se usan los empleados.

Permisos

El sistema permite asignar permisos indicando qué usuarios pueden acceder a los datos de qué obligados fiscales en una cuenta. Cada usuario sólo puede acceder a los datos de los obligados fiscales de su propia cuenta, nunca a los datos de obligados fiscales otras cuentas de cliente.

Para cada cuenta de cliente existen cuatro roles de usuarios: propietarios, colaboradores, empleados y lectores.

Para cada usuario se puede permitir o restringir selectivamente los obligados fiscales a los que tiene acceso.

Cada usuario tiene un rol único en la cuenta de cliente que es el mismo para todos los obligados fiscales a los que se le ha otorgado acceso.
