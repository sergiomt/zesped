¿Qué requisitos técnicos tiene zesped?

Para utilizar zesped se requiere:

  • Un ordernador personal PC-Windows o Mac-OSX
  • Un scanner con soporte Twain
  • Un navegador Firefox 17, Internet Explorer 9 de 32 bits o Google Chrome 24 *
  • El plugin de Java™ SE 6.0.260 para el navegador
  • Drivers Twain para el scanner utilizado
  • Un certificado digital de la FNMT

* Versiones anteriores o posteriores del navegador o del plugin de Java pueden presentar problemas de compatibilidad. Zesped no soporta IE 64 bits.

¿Cómo instalar Java en el navegador?

  1. Vaya a la página de descargas en Java.com.
  2. Descarge la versión de 32 bits de Java 6
  3. Una vez haya completado la descarga cierre el navegador
  4. Ejecute el archivo de instalación recien descargado

¿Cómo instalar el certificado en Internet Explorer o Google Chrome?

  1. Vaya a Panel de Control → Opciones de Internet.
  2. Seleccione la pestaña de Contenido y luego Certificados
  3. Utilice el botón de Importar para cargar su certificado en el almacén de certificados






¿Cómo instalar el certificado en Firefox?

  1. Escoja el menú de Herramientas → Opciones.
  2. Seleccione Avanzado → Cifrado → Ver Certificados
  3. Utilice el botón de Importar para cargar su certificado en el almacén de Mozilla
